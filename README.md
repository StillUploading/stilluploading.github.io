
# VinMate
<img width="128" height="128" alt="icon-128" src="https://github.com/user-attachments/assets/32e86146-d976-40e6-b839-a0cb6f3ae3f1" />

A Chrome extension that filters out **irrelevant listings** on Vinted. (This is an unofficial Vinted extension. Vinmate is not affiliated with, endorsed by, or sponsored by Vinted.)

**Installation:**

Go to chrome://extensions/, enable Developer Mode (top right corner), then drag and drop the .crx file onto the page.
